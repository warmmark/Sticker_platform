# Руководство пользователя

## Назначение

Программа предназначена для поиска исполнителей и оформления заказов на создание цифровых стикер-паков и эмоджи-паков.

## Запуск

1. Установить зависимости.
2. Выполнить миграции.
3. Создать администратора.
4. Запустить сервер.
5. Открыть сайт в браузере.

## Основные действия

### Регистрация

Пользователь вводит логин, email и пароль. После регистрации создаётся личный кабинет.

### Профиль

В профиле можно изменить аватар, никнейм, описание и ссылки на социальные сети.

### Объявления

В разделе «Мои объявления» пользователь создаёт предложение по стикерам или эмоджи. У одного пользователя может быть максимум два активных объявления: одно для стикеров и одно для эмоджи.

### Каталог

В каталоге можно искать объявления, фильтровать их по категории и цене, сортировать по новизне, популярности и цене.

### Избранное

Понравившиеся объявления можно сохранить в избранное.

### Заказы

Пользователь может оформить заказ по чужому объявлению. Заказы отображаются в активных и архивных.

### Сообщения

Пользователь может начать диалог с продавцом через страницу объявления.
