# Описание дипломного проекта

Тема: «Разработка платформы для создания и продажи цифровых стикеров и эмоджи-паков».

## Цель

Разработать веб-приложение, позволяющее художникам размещать предложения по созданию стикер-паков и эмоджи-паков, а заказчикам искать исполнителей, добавлять объявления в избранное, оформлять заказы и вести переписку.

## Объект разработки

Процесс поиска исполнителей и оформления заказов на создание цифровых стикеров и эмоджи.

## Предмет разработки

Веб-платформа с каталогом объявлений, личным кабинетом, заказами, избранным и личными сообщениями.

## Архитектура

Приложение построено как монолитное Django-приложение:

- `config` содержит настройки проекта;
- `marketplace` содержит модели, формы, представления, маршруты и шаблоны;
- SQLite используется как база данных;
- Django Templates используются для интерфейса;
- CSS-файл находится в `static/css/style.css`.

## Основные модели

- `Profile`: профиль пользователя;
- `Listing`: объявление;
- `Favorite`: избранное;
- `Order`: заказ;
- `Conversation`: диалог;
- `Message`: сообщение.

## Перспективы развития

- онлайн-оплата;
- рейтинг и отзывы;
- WebSocket-сообщения;
- загрузка файлов в заказах;
- интеграция с Telegram API;
- модерация объявлений.
